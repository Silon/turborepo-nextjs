## Getting Started

- Clone this repository or select "Use this template" on GitHub repository page
- Use `pnpm i` to install dependencies
- Use `pnpm prepare` to install husky hooks

## Commands

```sh
# Building apps
build

# Start development
dev

# Lint code with ESLint
lint

# Install husky hooks
prepare

# Format code with Prettier
format

# Commit with Commitizen (pretty git commit)
commit

# Inspect ESLint configuration
inspect-eslint
```

## Conventional Commits

We use [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) to standardize commit messages.
